## Quick Deployment Notes
- Deploy as static site.
- Ensure HTTPS is enabled.
- Set canonical domain to `https://www.capitalglassanddoor.com` or update canonicals if domain differs.
- Submit `sitemap.xml` in Google Search Console.
- Connect form handling to CRM/email webhook for production use.
