Property Manager Emergency Playbook for Glass and Door Incidents

Property teams need repeatable response workflows. This playbook outlines a simple model that scales across multiple sites.

Portfolio Response Workflow

  1. Confirm safety and immediate site-secure requirements.
  2. Notify tenants and site contacts with short status update.
  3. Dispatch emergency secure-up and collect scope notes.
  4. Approve replacement plan with access and timing windows.
  5. Request post-job documentation for records and claims.

Key Metrics to Track

  • Time to initial dispatch contact.
  • Time to secure opening.
  • Time to replacement completion.
  • Repeat-failure causes by opening type.
